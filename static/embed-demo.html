‚öôÔ∏è Configuration

Customize the widget by adding configuration before loading the script:

<script> // Configure widget before loading window.COLLEGE_BUDDY_API = 'https://your-api-domain.com'; window.COLLEGE_BUDDY_THEME = 'indigo'; window.COLLEGE_BUDDY_POSITION = 'bottom-right'; window.COLLEGE_BUDDY_AUTO_OPEN = false; window.COLLEGE_BUDDY_GREETING = 'Hi! How can I help you?'; </script> <script src="https://yourdomain.com/static/widget.js"></script>

Configuration Options

Option Type Default Description
COLLEGE_BUDDY_API string localhost:8001 API server URL
COLLEGE_BUDDY_THEME string indigo Color theme
COLLEGE_BUDDY_POSITION string bottom-right Widget position (bottom-right, bottom-left, top-right, top-left)
COLLEGE_BUDDY_AUTO_OPEN boolean false Auto-open widget on page load
COLLEGE_BUDDY_GREETING string "Hi! How can..." Custom greeting message

üîå JavaScript API

Control the widget programmatically:

// Open the widget CollegeBuddyWidget.open(); // Close the widget CollegeBuddyWidget.close(); // Toggle widget state CollegeBuddyWidget.toggle(); // Check if widget is open if (CollegeBuddyWidget.isOpen()) { console.log('Widget is currently open'); }

Example: Open widget on button click

<button onclick="CollegeBuddyWidget.open()"> Need Help? Chat with us! </button>
